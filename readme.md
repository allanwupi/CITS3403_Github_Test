# CITS3403 Group Project - Checkpoint 1

This repository was made for us to test Git commands. Clone this repository to your local machine by running `git clone https://github.com/allanwupi/CITS3403_Github_Test.git`.

The basic work flow for branching and developing new features:

```bash
git checkout main
git pull origin
git checkout -b <newBranch>
git commit -m "descriptive message"
git push origin <newBranch>
```

After this, create a pull request in GitHub, have someone review the code and approve the merge into main.
